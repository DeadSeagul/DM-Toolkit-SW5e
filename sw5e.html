ü§ñ AI Setup Guide (Click to Expand)

This toolkit uses Google's Gemini AI. It is free to use, but you need to get your own "Key" from Google.

Step 1: Get a Key

Go to Google AI Studio and click Create API Key.

Step 2: Create Project

If asked, select "Create API key in a new project". Give it a name like "DM Tool".

Step 3: Copy & Save

Copy the long code starting with AIza....

Go to the Prep Tab, paste it into the "AI Configuration" box, and click Save.
